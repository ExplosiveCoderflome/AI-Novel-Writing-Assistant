# docs/superpowers

存放通过 Claude superpowers 工作流(如 `writing-plans`、`subagent-driven-development` 等 skill)产出的执行计划与方案稿。

## 子目录

- `plans/` — 多步任务的实施计划稿,文件名格式 `YYYY-MM-DD-<feature>.md`

## 与 docs/plans/ 的区别

- `docs/plans/` 是项目主干长期维护的计划集
- `docs/superpowers/plans/` 是单次任务的工作稿;完成后可保留作历史参考,也可挑选合并到 `docs/plans/`
