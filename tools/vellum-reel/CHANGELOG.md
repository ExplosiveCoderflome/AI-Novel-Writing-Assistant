# Changelog

## 0.3.0

- 品牌升级为 VellumReel / 卷影，并增加全片品牌框架与阅读进度。
- 增加镜头强度、拉远、上移、调色与转场配置。
- 增加长篇叙事节拍卡、题记数组、日志线和片尾 CTA。
- 增加旁白、对话、引用三种字幕语气与关键词强调。
- 默认工程改为无外部媒体依赖的原创公开示例。
- 私有书稿、旁白、图片、成片和供应商资产地址默认排除在 Git 之外。
- 增加 MIT License、第三方许可说明、素材权利指南和 GitHub Actions。

## 0.2.0

- 增加音频时长驱动的视频时长与分镜自动同步。
- Whisper.cpp 词级结果会按中文字数、句读、停顿和最长展示时间自动合并。
- 增加 `produce:preview` 和 `produce:final` 一键生产流程。
- 正式渲染在没有旁白时默认中止，防止静音样片被误当成片。
- 验片时输出 `out/qc/report.json`，并强制检查时长、9:16 画幅和 H.264 编码。
- 增加字幕分屏和时间轴缩放的单元测试。

## 0.1.0

- 首个可运行的 Remotion 竖屏图书号模板。
- ElevenLabs 旁白、Whisper.cpp 对齐、SRT 导入和字幕估时入口。
- FFmpeg/ffprobe 成片检查和抽帧联系表。
